# Cifrado César

#Empleados bancarios deben cifrar de manera manual la información de los nuevos usuarios de tarjeta crédito, indicnando datos como nombre, ciudad y clave inicial, para esto utilizan un cifrado a mano que retrasa de manera exagerada sus tiempos de trabajo, con esta aplicación ellos podrán asignar el mismo mensaje de una manera más rápida y compartir el mismo al cliente para que el pueda descifrar su información.
